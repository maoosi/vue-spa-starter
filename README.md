# vuejs-template-spa

> The last Vue SPA boilerplate your 🐱 will ever need.

## Requirements

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### Installation

```bash
sao maoosi/vuejs-template-spa my-project
```
